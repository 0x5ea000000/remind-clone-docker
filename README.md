# remind-clone-docker

required files to run remind-clone in docker

## How to use
1. rename backend, frontend folders to back-end, front-end
2. add source code from back-end, front-end into 2 above folders.

```
remind-clone/
├─ front-end/
│  ├─ Dockerfile
│  ├─ source files
│  ├─ .dockerignore
├─ back-end/
│  ├─ Dockerfile
│  ├─ source files
│  ├─ .dockerignore
├─ docker-compose.yaml

```
